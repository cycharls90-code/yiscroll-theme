# YiScroll — Custom WordPress Theme

A premium, conversion-focused custom WordPress theme built for YiScroll Digital Marketing Agency, Coimbatore.

## Tech Stack

- PHP (WordPress)
- HTML5 + CSS3 (no frameworks)
- Vanilla JavaScript
- Google Fonts: Syne + DM Sans
- Remix Icons CDN

## Local Setup

1. Install [LocalWP](https://localwp.com/)
2. Create a new site in LocalWP
3. Clone this repo into: `wp-content/themes/yiscroll/`
4. Go to WP Admin → Appearance → Themes → Activate **YiScroll**
5. Go to Appearance → Menus → Create Primary Menu with: Home, Services, Portfolio, About
